🌸 关于本站

这是 mako 设计的个人学习记录网站，用于记录课程笔记、管理日程、展示作业成果。

网站使用纯原生 HTML + CSS + JavaScript 构建，数据保存在您的本地浏览器中（localStorage）。所有功能无需注册即可使用！

🚀 快速上手

  1. 设置您的 GitHub 仓库（可选）
    在下方「⚙️ 自定义设置」中填入您的 GitHub 仓库路径，这样上传文件时会显示正确的路径提示。
  2. 添加您的课程
    在首页点击「➕ 添加课程」，填写课程名称、选择图标即可。
  3. 开始记录笔记
    进入「课程笔记」页面，选择课程后即可使用 Markdown 语法编写笔记。
  4. 管理作业和待办
    在「课程作业」添加作业，带截止日期的作业会自动同步到日程的待办列表。
  5. 自定义背景图
    在下方「🖼️ 更换背景图」上传本地图片作为网站背景。

🧭 页面功能

  • 📚 首页 - 展示所有课程卡片，点击可编辑课程信息或查看详情
  • 📝 课程笔记 - 按课程分类编写 Markdown 笔记，支持文件上传
  • 📋 课程作业 - 添加/管理各课程的作业，带截止日期自动同步到待办
  • 📅 日程 - 上方是待办事项列表，下方是可编辑的课程表
  • ❓ 帮助 - 本页面

📅 待办事项使用指南

  • 输入待办内容，选择截止日期（可选）和关联课程，点击「添加」
  • 点击复选框可标记完成，完成后文字会变灰并划线
  • 截止日期颜色：红色=已过期，橙色=今天，黄色=3天内
  • 筛选按钮：「全部」「今天」「即将到期」「已完成」
  • 添加带截止日期的作业时，会自动在待办中创建关联任务

📊 课程表使用指南

  • 点击「➕ 添加课程项」可以添加新课程到课表中
  • 选择课程、星期、节次、周次类型（全部/单周/双周）和上课地点
  • 点击右上角的「⚙️ 设置」可以修改学期开始日期和总周数
  • 点击「本周」按钮可快速回到当前周
  • 点击时间节次（左侧）可以编辑该节次的名称和时间
  • 单周/双周课程会根据当前周自动显示/隐藏

📎 文件上传说明

在「课程笔记」页面可以上传文件（Markdown、文本、PDF）。

  • 上传后请根据提示的 GitHub 路径，将文件手动上传到您的仓库对应目录
  • Markdown 和文本文件内容会保存在浏览器中，可直接预览
  • PDF 文件只记录元数据，实际预览需要上传到 GitHub 后点击「在新窗口打开」

❓ 常见问题

Q: 数据存储在哪里？

A: 所有数据保存在您浏览器的 localStorage 中。清除浏览器数据会导致笔记、作业、课程表等信息丢失。建议定期备份重要数据。

Q: 笔记会自动保存吗？

A: 不会自动保存，需要手动点击「💾 保存」按钮。关闭页面时会提示是否保存未保存的更改。

Q: 如何自定义网站外观？

A: 在下方「🖼️ 更换背景图」上传本地图片作为背景。在「⚙️ 自定义设置」中配置 GitHub 仓库路径。

Q: 如何获取自己的网站副本？

A: 将本项目Fork到您的 GitHub 仓库，修改内容后通过 GitHub Pages 托管即可。

Q: 作业和待办有什么区别？

A: 作业是课程相关的任务，待办是更通用的任务清单。添加带截止日期的作业时，系统会自动创建关联的待办事项。
