🌸 关于本站

这是 mako 的个人学习记录网站，用于记录课程笔记、管理日程、展示作业成果。

网站使用纯原生 HTML + CSS + JavaScript 构建，无任何外部框架依赖。数据保存在本地浏览器中。

🧭 页面导航

  • 首页 - 展示所有课程及学习进度，可添加新课程
  • 课程笔记 - 按课程分类记录 Markdown 笔记，支持文件上传
  • 课程作业 - 查看和添加各课程的作业，记录完成状态
  • 日程 - 管理待办事项和课程表
  • 帮助 - 本页面，提供使用指南

⌨️ 快捷操作

导航栏 点击顶部导航栏可快速切换页面
课程卡片 点击课程卡片可进入课程详情或开始学习
笔记保存 编辑笔记后点击"💾 保存"按钮保存到本地
笔记切换 使用"✏️ 编辑"和"👁️ 预览"切换笔记视图
作业链接 点击"查看代码"可跳转到 GitHub 仓库
添加课程 在首页点击"➕ 添加课程"按钮可添加新课程
添加待办 在日程页面输入内容后按 Enter 或点击"添加"按钮
编辑课程表 直接在课程表单元格中输入内容，自动保存
添加作业 在作业页面点击"➕ 添加作业"按钮
完成作业 点击作业卡片上的复选框，作业和待办同步完成

❓ 常见问题

Q: 笔记会自动保存吗？

A: 笔记不会自动保存，需要手动点击"💾 保存"按钮。关闭页面时会提示是否保存。

Q: 笔记数据存储在哪里？

A: 笔记存储在浏览器的 localStorage 中，每个课程的笔记独立保存。清除浏览器数据会丢失笔记。

Q: 如何添加新的作业？

A: 在作业页面点击"➕ 添加作业"按钮，填写作业标题、选择课程、设置截止日期即可。

Q: 作业和待办有什么关系？

A: 当你添加带截止日期的作业时，系统会自动在待办事项中创建一个关联任务。完成其中一个，另一个也会同步完成。

Q: 如何添加新的课程？

A: 在首页点击"➕ 添加课程"按钮，填写课程名称、图标、学分等信息即可。

Q: 待办事项的截止日期颜色是什么意思？

A: 红色表示已过期，橙色表示今天到期，黄色表示即将到期（3天内）。

Q: 上传的文件存储在哪里？

A: Markdown 和文本文件内容存储在 localStorage 中，PDF 仅记录元数据。如需 PDF 预览，请将实际 PDF 文件放入 pdf/ 目录。
