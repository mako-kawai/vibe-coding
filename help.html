🌸 关于本站

这是 mako 的个人学习记录网站，用于记录课程笔记、展示作业成果。

网站使用纯原生 HTML + CSS + JavaScript 构建，无任何外部框架依赖。

🧭 页面导航

  • 首页 - 展示所有课程及学习进度
  • 课程笔记 - 按课程分类记录 Markdown 笔记
  • 课程作业 - 查看各课程的作业及完成状态
  • 帮助 - 本页面，提供使用指南

⌨️ 快捷操作

导航栏 点击顶部导航栏可快速切换页面
课程卡片 点击课程卡片可进入课程详情或开始学习
笔记保存 编辑笔记后点击"💾 保存"按钮保存到本地
笔记切换 使用"✏️ 编辑"和"👁️ 预览"切换笔记视图
作业链接 点击"查看代码"可跳转到 GitHub 仓库

❓ 常见问题

Q: 笔记会自动保存吗？

A: 笔记不会自动保存，需要手动点击"💾 保存"按钮。关闭页面时会提示是否保存。

Q: 笔记数据存储在哪里？

A: 笔记存储在浏览器的 localStorage 中，每个课程的笔记独立保存。清除浏览器数据会丢失笔记。

Q: 如何查看作业的代码？

A: 点击作业卡片上的"查看代码"按钮，会跳转到 GitHub 仓库页面。

Q: 为什么作业显示"待开始"？

A: 这表示该作业尚未开始，通常是因为对应的课程模块还未学习到。
