模块七：视频即代码

4学时

📖 模块介绍

用 Remotion 以代码方式制作多媒体视频，让视频可编程。本模块将带你探索用代码创建视频的全新方式，实现视频内容的程序化生成。

🎯 学习目标

  • 了解 Remotion 的基本概念
  • 学会用 React 代码创建视频
  • 实现动态数据驱动的视频生成
  • 制作可复用的视频模板

📚 学习内容

🎞️ Remotion
🎥 代码视频
🎨 多媒体
🔄 动态生成

💻 实践任务

使用 Remotion 创建一个动态视频，可以是数据可视化视频、个人介绍视频等，支持数据驱动的自动更新。
