模块二：网页开发基础

2学时 · 线下 · 🎨 前端入门

📖 模块介绍

学习 HTML/CSS/JavaScript 基础，搭建个人课程主页。本模块将带你入门 Web 开发的三件套：HTML 负责结构、CSS 负责样式、JavaScript 负责交互，为后续的网页进阶开发打下基础。

🛠️ 核心工具

📝 VS Code
🏗️ HTML5
🎨 CSS3
⚡ JavaScript
🚀 GitHub Pages

📋 产出成果

🏠

个人课程主页

包含个人介绍、课程进度的完整主页

📱

响应式页面

适配手机和电脑的页面布局

🗺️ 学习路径

1

HTML 基础结构

学习常用标签：div、h1-h6、p、a、img、ul/ol

2

CSS 样式美化

学习选择器、盒模型、Flexbox、响应式布局

3

JavaScript 交互

添加简单交互：点击事件、动画效果

4

部署上线

上传至 GitHub Pages，获得可访问的网址

🎯 学习目标

  • 掌握 HTML 基础标签与文档结构
  • 理解 CSS 选择器与盒模型
  • 学会使用 JavaScript 处理简单交互
  • 完成个人课程主页的搭建与部署

💡 关键概念

🏗️ HTML 结构
📦 CSS 盒模型
📐 Flexbox 布局
📱 响应式设计
