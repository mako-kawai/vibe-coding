# 个人学习主页 - 规范文档

## 1. 项目概述

- **项目名称**：个人学习主页
- **项目类型**：静态网站（HTML + CSS + JavaScript）
- **核心功能**：展示个人简介、课程列表、作业链接
- **目标用户**：氛围编程课程学员

## 2. 设计风格

- **风格**：日漫风格（Anime-inspired）
- **主题**：清新、柔和、带有日式美学元素
- **配色方案**：
  - 主色：樱花粉 `#FFB7C5`
  - 背景：米白/浅灰 `#FAFAFA`
  - 强调色：薄荷绿 `#98D8C8`
  - 文字：深灰 `#2D2D2D`
- **字体**：使用 Noto Sans JP（中文/日文）+ 无衬线英文字体
- **装饰元素**：樱花花瓣飘落动画、柔和的阴影与圆角

## 3. 页面结构

### 3.1 导航栏
- 固定在顶部
- 包含：Logo/标题、导航链接（课程、简介、模块、作业）
- 背景半透明，带模糊效果

### 3.2 课程介绍区
- 课程名称：氛围编程
- 课程代码：PSYCH280 / COMS20003A
- 课程特色：不写代码、没有考试、作业会做就行、氛围学习
- 授课信息：教师、学分、学时、授课方式、时间地点、容量
- 考核方式：课程项目 60%、课堂参与 40%

### 3.3 个人介绍区
- 头像区域（圆形，带柔和边框）
- 姓名与昵称
- 一段简短的自我介绍
- 社交链接（GitHub 主页链接）

### 3.4 课程模块区
- 卡片式布局
- 10个课程模块，每项显示：模块名称、描述、学时、线下标识、状态
- 悬停时有轻微上浮效果

### 3.5 作业展示区
- 网格/列表布局
- 每项作业显示：作业标题、简介、完成状态、GitHub 链接按钮
- 点击按钮跳转至 GitHub 仓库

### 3.6 页脚
- 版权信息
- 简洁的装饰

## 4. 交互效果

- **樱花飘落动画**：页面背景有缓慢飘落的樱花花瓣
- **卡片悬停**：轻微上浮 + 阴影加深
- **按钮交互**：悬停时颜色变化 + 轻微缩放
- **平滑滚动**：导航点击平滑滚动到对应区域
- **入场动画**：页面加载时元素渐入

## 5. 技术实现

- **HTML5**：语义化标签
- **CSS3**：Flexbox、Grid、动画、响应式
- **JavaScript**：樱花动画、导航交互、滚动效果
- **无外部依赖**：纯原生实现

```
/

├── styles.css      # 样式文件
├── script.js       # 交互脚本
└── SPEC.md         # 规范文档
```
