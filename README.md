## ✨ 功能特色

### 📚 课程管理
- 展示个人课程列表
- 点击进入课程详情页
- 查看课程进度

### 📝 课程笔记
- 支持 Markdown 格式
- 按课程独立保存
- 编辑/预览切换
- 本地存储，不丢失

### 📋 作业展示
- 按课程分组
- 显示完成状态
- 链接到 GitHub 仓库

### 🎨 设计风格
- 樱花粉配色 (#FFB7C5)
- 薄荷绿点缀 (#98D8C8)
- 樱花飘落动画
- 柔和阴影与圆角

## 🚀 快速开始

### 查看网站
直接用浏览器打开 `index.html` 即可。

### 本地开发
推荐使用 VS Code 的 Live Server 扩展：
1. 安装 Live Server 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## 💡 使用技巧

### 笔记
- 选择课程后开始记录
- 使用 Markdown 格式
- 点击保存按钮或关闭页面时自动保存

### 作业
- 点击"查看代码"跳转到 GitHub 仓库
- 使用标签切换不同课程的作业

## 🚀 部署到 GitHub Pages

本项目已配置 GitHub Pages，可以在线访问！

### 启用 GitHub Pages：
1. 进入仓库 **Settings** > **Pages**
2. Source 选择 **Deploy from a branch**
3. Branch 选择 **main**，文件夹选择 **/ (root)**
4. 点击 **Save**
5. 等待 1-2 分钟，网站将在 https://mako-kawai.github.io/vibe-coding/ 上线

### 更新网站：
推送代码到 main 分支后，GitHub Pages 会自动更新。
